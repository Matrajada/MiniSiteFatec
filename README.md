# 🌐 Mini Site FATEC - Baixada Santista Rubens Lara

Este projeto consiste em um mini site institucional desenvolvido com HTML e CSS, apresentando informações detalhadas sobre a **FATEC Baixada Santista Rubens Lara**.

## 📌 Descrição

O site foi desenvolvido com o objetivo de praticar a construção de páginas web completas, utilizando estrutura semântica, navegação interna e estilização externa.

A página reúne diversas informações institucionais, como:

- Cursos oferecidos
- Distribuição de vagas por turno
- História da instituição
- Missão institucional
- Contato e localização
- Calendário acadêmico

## 🚀 Funcionalidades

- 🧭 Navegação interna com âncoras (ex: "voltar ao topo", "voltar à tabela")
- 🖼️ Imagens clicáveis com redirecionamento para o site oficial
- 📊 Tabela organizada com distribuição de vagas por curso
- 🔗 Links internos e externos
- 📱 Estrutura preparada para responsividade (`meta viewport`)
- 🎨 Estilização externa com CSS (`main.css`)
- ⚡ Lazy loading em imagens para melhor performance

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
